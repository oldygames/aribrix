## Versioning

Version format: `vN.M`
- `N` is the [LIBWO](https://github.com/oldygames/libwo) version in use
- `M` is a sequentially increasing integer; for each new `N`, `M` starts at `0`

Examples: `v1.0`, `v1.1`, `v2.0`.
